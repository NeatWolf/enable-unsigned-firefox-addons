# Code of Conduct

Keep discussion focused, factual, and respectful.

This repository is provided as-is. The code of conduct does not create a support
commitment, and it does not make GitHub issues or pull requests a help desk for
local Firefox installs.

For contributions, stay on the narrow script and repository maintenance scope:
safe path handling, rollback behavior, verification, documentation, and
compatibility with the prerequisites listed in `README.md`.
